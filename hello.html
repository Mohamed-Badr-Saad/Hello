Git Notes::

* git clone 'URL' : to download the repository to my computer.

* git add 'file_name.ext' : to add the file(to something like a queue) that I want to save(commit) it to my repository.

* git commit -m "message" : to save the added files(to something like a queue) to my local repository on computer.

* git push : to save(push) changes thay have been commited to the online repository on GitHub.

* git commit -am "message" : to add all the files that have changed and commit them in 1 step.

* git pull : to download the most recent update(changes) that have been comitted to the repository.

* git status : tells us what is currently happening inside our repository.
